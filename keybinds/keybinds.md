## CODING ##

```
## Basic ##
shift+?: global search
ctrl+n: toggle file tree
ctrl+shift+n: go to file tree
ctrl+{num}: switch to view
ctrl+/: go to file
ctrl+w: close tab
ctrl+t: toggle terminal


## LLM ##
ctrl+l: toggle chat window
ctrl+shift+l: go to chat window
alt+l: new chat
ctrl+alt+l: toggle conversations
ctrl+k: inline edits

## Misc defaults ##
ctrl+m ctrl+s: keyboard (s)hortcuts

```
## Vim ##
```
ctrl+i: undo last hop
ctrl+o: redo last hop 
ctrl+a: increment number at cursor
ctrl+x: decrement under cursor
```
## OS ##
```
super: search for program
super+{num}: go to {num} workspace
super+q: quit window
super+t: open terminal
super+o: rotate tiling window
super+{hjkl}: move to up down left right window
super+y: toggle tiling

(TODO) super+?: open keybinds
```
## Browser ##
```
ctrl+t: new tab
ctrl+w: close tab
ctrl+l: go to search bar
```

## Obsidian ##
```
ctrl+n: (coding)
ctrl+t: (browser)
ctrl+enter: open link under cursor in current tab
alt+enter: open link under cursor in new tab
ctlr+alt+enter: open link under cursor to the right
```
